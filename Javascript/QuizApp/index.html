Introduction to Quiz App Project

The Quiz App is a web-based application developed as part of my learning in web development. It allows users to input, fetch, search, and display quiz questions efficiently. The app provides the following key features:

Key Features

  • Input and Store Quiz Questions: Users can submit quiz questions, which are then stored for future use.
  • Fetch Stored Questions: Pre-existing quiz questions can be fetched and displayed.
  • Search Functionality: Users can search quiz questions based on their content.
  • Question Display: Users can view all quiz questions entered or fetched, with the ability to sort them in ascending or descending order.
  • Game Mode: The app offers a competitive quiz game for two players, where the winner is determined based on the score, with the maximum score calculated as half the total number of questions.

Enhanced UI Features

  • Coloring Input Fields: Correct answers are highlighted before submission, providing visual feedback.
  • Shuffling Answers: The order of multiple-choice answers is shuffled for each quiz, ensuring fairness.

The app also features a simple UI design, focused on ease of use and providing an intuitive experience for users. It incorporates modern web development techniques, including responsive design, dynamic content updates, and asynchronous data fetching to create an interactive and engaging platform for quiz enthusiasts.
